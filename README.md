чтобы запускать скрипт нужно сперва дать право исполнения через команду:
```
chmod +x main.sh //предоставление права на исполнение
./main.sh //запуск скрипта
```
или можно запускать через bash:
```
bash main.sh
```
Результат: вы получите текстовый файл word_documents.txt в формате:
```
{word_count} {file_name}
{symbol_count} {file_name}
```
