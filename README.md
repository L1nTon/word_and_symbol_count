Чтобы запустить скрипт, сначала нужно выдать ему право на исполнение:
```
chmod +x main.sh //предоставление права на исполнение
./main.sh //запуск скрипта
```
Также можно запускать его через bash:
```
bash main.sh
```
Результат:
Будет создан файл word_documents.txt со строками вида:
```
{word_count} {file_name}
{symbol_count} {file_name}
```
